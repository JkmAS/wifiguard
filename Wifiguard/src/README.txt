If you want to use Wifi guard, you can download a prepared package on the web wifiguard.cz

Wifi Guard is a small GUI for arp-scan. 

To run the Wifi Guard it is necessary put the downloaded package with app into "user / home". 
You can run the application by clicking directly on the application launcher with the WiFi Guard icon (desktop file).

It is necessary to have installed arp-scan:
	Install arp-scan in Ubuntu
	Open the terminal and run the following command
	
	sudo apt-get install arp-scan
