JkmAS Wifi Guard Application

Wifi Guard is a small tool for scanning Wifi network using the ARP command, which displays network devices and information displayed in a user-friendly Graphical User Interface. The application is very simple and because of ARP scanner very fast.

Function:

Buttons

  • Start

    When you start monitoring the network. The application shows you all the devices on the network, their IP address and other data. All information is stored in the database.

    The scanning speed depends on your internet connection speed.

  • Stop

    If for some reason you want to stop monitoring network, just press stop button.

  • Settings

    Here you can view or delete the database of found devices.

  • Show

    Show in the browser list of devices and basic information about each of them.
